# express-rate-limit-redis

Robust rate limiting middleware for Express.js using Redis.

## Features
- 🚀 Fast and optimized
- 🔒 Secure by default
- 🛠 Easy to configure

## Installation
```bash
git clone https://github.com/HafezMosleh/express-rate-limit-redis.git
cd express-rate-limit-redis
```
