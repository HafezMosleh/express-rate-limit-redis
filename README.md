## 🌟 Overview
Protect your APIs from DDoS attacks and brute forcing. This project implements a scalable rate-limiting architecture for Express.js applications, utilizing Redis to track requests across multiple Node.js instances flawlessly.

## ✨ Key Features
- 🛡️ **DDoS Protection**: Effectively throttles excessive requests.
- ⚖️ **Cluster Ready**: Uses Redis to sync rate limits across load balancers.
- ⏱️ **Customizable Windows**: Easily define limit rules per route or globally.

## 🚀 Getting Started

### Prerequisites
Make sure you have `Node.js_Redis` installed on your local machine.

### Installation
```bash
git clone https://github.com/HafezMosleh/express-rate-limit-redis.git
cd express-rate-limit-redis
npm install
```

### Usage
```bash
node main.js
```
